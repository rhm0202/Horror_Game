# ROOM 13 (가칭)

> 1인칭 공포 방탈출 게임 — Unity / Windows PC

---

## 게임 소개

눈을 떠보니 낯선 방 안이었다. 문은 잠겨 있고, 자신이 누구인지조차 기억이 없다.  
손전등 하나에 의지해 어둠 속을 탐색하고, 단서를 수집하며 기믹을 해결해 탈출구를 찾아라.

| 항목 | 내용 |
|------|------|
| 장르 | 1인칭 공포 방탈출 (First-Person Horror Escape) |
| 플랫폼 | Windows (PC) |
| 엔진 | Unity 6.3 LTS |
| 목표 플레이 시간 | 15 ~ 20분 |
| 레퍼런스 | Outlast, Amnesia: The Dark Descent, Layers of Fear |

---

## 게임플레이

- 중앙 복도와 3개의 방을 1인칭 시점으로 탐색
- 각 방의 아이템과 단서를 수집해 기믹 해결
- 유령은 복도만 배회 → **방 안은 안전 구역 / 복도는 위험 구역**
- 3개의 회로 부품(전원 모듈, 퓨즈, 전선)을 모아 전력 복구 후 출입구 탈출

### 주요 아이템

| 아이템 | 획득 위치 | 용도 |
|--------|-----------|------|
| 손전등 | 방 1 | 기본 장비, 켜고 끄기 가능 |
| 열쇠 | 방 1 서랍 | 복도로 나가는 문 개방 |
| 전원 모듈 | 방 1 | 회로 장치 부품 |
| 부적 | 방 2 | 유령 접촉 시 1회 자동 보호 |
| 퓨즈 | 방 2 보관함 기믹 | 회로 장치 부품 |
| 카드키 | 방 2 | 방 3 전기실 입장 |
| 전선 | 방 3 전기 장치 기믹 | 회로 장치 부품 |
| 정체불명의 쪽지 | 방 3 | 출입구 키패드 비밀번호 힌트 |

## 프로젝트 구조

```
Assets/
├── _Game/                        # 실사용 리소스
│   ├── Scripts/
│   │   ├── Player/               # 플레이어 이동, 인벤토리
│   │   │   └── Flashlight/       # 손전등 on/off, UV 모드
│   │   ├── Item/                 # 아이템 데이터, 줍기
│   │   ├── Interactable/         # 문, 서랍, 키패드 등
│   │   ├── Enemy/                # 유령 AI
│   │   ├── Manager/              # 게임 매니저
│   │   └── UI/                   # 소지품 UI, 메시지 안내
│   ├── Items/                    # 아이템 ScriptableObject
│   ├── Sprites/
│   │   ├── Items/                # 아이템 아이콘
│   │   └── UI/                   # 손전등 상태 아이콘 등 UI 이미지
│   ├── Scenes/                   # 게임 씬
│   ├── Prefabs/                  # 프리팹
│   ├── Materials/                # 머티리얼
│   └── Audio/                    # BGM, SFX
└── ThirdParty/                   # 외부 다운로드 에셋
    ├── Dnk_Dev/                  # Hospital Horror Pack
    ├── Electric Torch/           # Electric Torch
    └── HorrorPuzzleItems/        # Horror Puzzle Items
```
