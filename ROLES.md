# 역할 분담

## 류 — 플레이어 · 게임 흐름 · 기믹

### 이미 구현됨
- 플레이어 이동 (WASD), 마우스 시점
- 손전등 켜고 끄기 (E), UV 모드 (R)
- 인벤토리 (아이템 소지 목록, 아이템 선택)
- 아이템 줍기 (F키 레이캐스트)

### 구현 필요
**아이템 사용 시스템**
- 인벤토리에서 아이템 선택 후 오브젝트에 F키로 사용
- 올바른 아이템 사용 시 결과 발생, 아닌 경우 안내 메시지 출력

**기믹**
- 방 2 잠긴 보관함 기믹 (힌트 기반 해결 → 퓨즈 획득)
- 방 3 전기 장치 기믹 (해결 → 전선 획득)
- 방 3 회로 장치 (전원 모듈 + 퓨즈 + 전선 순서대로 장착 → 전력 복구, 키패드 활성화)
- 복도 출입구 키패드 (비밀번호 입력 → 탈출 성공)
- 일지 조사 스크립트 (텍스트 팝업)

**게임 흐름**
- 방 1 클리어 시 유령 순찰 시작 트리거
- 회로 장치 복구 후 유령 이동 속도 증가 트리거
- 게임오버 / 클리어 씬 전환

## 맵 담당 — 맵 제작 · 인터랙터블 · 사운드

### 구현 필요
**씬 레이아웃**
- 중앙 복도 (복도 끝에 출입구 배치)
- 방 1 — 감금실 (서랍, 상자/선반, 잠긴 문, 일지 오브젝트)
- 방 2 — 보관실 (서랍, 잠긴 보관함, 책상, 일지 오브젝트)
- 방 3 — 전기실 (전기 장치, 낡은 책상/게시판, 회로 장치, 일지 오브젝트)

**인터랙터블 스크립트**
- 서랍 열고닫기 (방 1 열쇠 획득 / 방 2 부적 획득)
- 문 열기 (방 1 잠긴 문 → 열쇠 사용 / 방 3 전기실 문 → 카드키 사용)
- 상자·선반 조사 (아이템 획득)

**오브젝트 배치**
- 아이템 프리팹 씬 배치 (손전등, 열쇠, 전원 모듈, 퓨즈, 카드키, 전선, 부적, 쪽지)
- 각 방 조명 배치 (깜박이는 불빛 포함)

**사운드**
- 각 방·복도 환경음 AudioSource 배치
- 문 열림·서랍 열림 SFX 연결
- BGM 오디오소스 씬 배치

## 유령 AI 담당

### 구현 필요
- 중앙 복도 순찰 (방 1 클리어 트리거 이후 활성화)
- 플레이어 탐지 범위 진입 시 추격 상태 전환
- 플레이어가 방 안으로 들어가거나 탐지 범위 벗어나면 순찰 복귀
- 유령이 방 안으로 진입하지 않도록 제한
- 접촉 시 GameManager.SetState(GameOver) 호출
- 부적 소지 시 접촉 1회 보호 → 이후 분노 상태 (순찰 범위 확장)
- 회로 복구 후 이동 속도 증가 (GameManager 상태 변경 감지)
